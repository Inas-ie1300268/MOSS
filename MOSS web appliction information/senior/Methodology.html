Methodology


The methodology of the online shopping system includes the following components:
  1. Data Retrieval:
    It sends queries using APIS to retrieve data from Amazon and eBay and integrates the results in a uniform structure.
  2. Data Discretization:
    It discretizes data into ranges based on certain formulas to improve data analysis.
  3. Conceptual Entity Extraction:
    It extract entities from retrieved data based on the formal concept analysis theory.
  4. Conceptual Tree Browser:
    The extracted concepts are used to generate a conceptual tree for browsing the retrieved data in an organized and easy way.
  5. Association Rules Generation:
    It uses the Apriori algorithm to generate rules that help users to reformulate the query in an easy way.
